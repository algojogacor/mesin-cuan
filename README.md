## The Vision

> *"This isn't a bot. It's an AI Video Architect that works around the clock вҖ” discovering trends, writing scripts, rendering cinematic footage, and uploading to YouTube, fully automated."*

**Mesin Cuan Viral Architect** is a 24/7 AI-powered YouTube content production system. It's not merely a script; it's an **autonomous video factory** that understands the algorithm, detects trends before they peak, and executes every frame with cinematic precision.

Imagine having a complete production team вҖ” trend researcher, scriptwriter, voice actor, video editor, and upload manager вҖ” working without rest, without salary, without complaints. That's Mesin Cuan.

## вҡҷпёҸ 12-Engine Architecture

### рҹ“Ў Viral Loop Engine
Real-time trend intelligence combining **YouTube Data API v3**, **Google Trends**, and **YouTube Search Suggestions** вҖ” refreshed every 6 hours.

### рҹ§  Dual Parallel AI Scripting
**Qwen + Ollama generate scripts in parallel**, then cross-score each other (anti-sycophancy). Highest-scoring script proceeds to production. Fallback chain: Ollama вҶ’ Qwen вҶ’ Groq.

### рҹҺҜ Script Quality Scorer
Multi-dimensional scoring: **hook strength**, **curiosity gap**, **pacing**, **emotional impact**, **retention prediction** вҖ” scripts below threshold are auto-rewritten.

### рҹ”— Hook Engine
Auto-injects opening hooks that maintain viewer retention in the first 3 seconds вҖ” driven by per-channel retention analytics.

### рҹ“ҡ Memory Engine
Tracks previously produced topics вҖ” prevents content repetition and ensures variety across videos.

### рҹ”Қ Research Engine
Automated web research before script generation вҖ” ensures factual accuracy and up-to-date context.

### рҹ“ә Series Engine
Auto-generates multi-part content (Part 1, 2, 3...) with cliffhanger logic and cross-episode callbacks.

### рҹҢҹ Neon Visuals
Cinematic FFmpeg renderer: **glowing neon text**, **glassmorphism panels**, **cinematic letterbox** + vignette + color grading.

### рҹ”Ҡ Smart SFX Mixer
Niche-aware sound effects: Horror вҶ’ heartbeat/thunder, Psychology вҶ’ mind-tone/focus hum, Motivation вҶ’ crowd cheer/stadium echo.

### рҹ–јпёҸ Thumbnail Intelligence
AI-driven thumbnail text & style selection based on niche and topic вҖ” CTR optimization.

### рҹ“Ҡ OAuth2 Analytics
**YouTube Analytics API v2** integration вҖ” per-channel retention dashboard, drop-off analysis, content optimization.

### вҡЎ Pipeline Estimator
ETA prediction for multi-channel batch renders вҖ” accurate estimation before execution.

## рҹҡҖ Installation & Setup

### Step 1 вҖ” Clone & Environment

```bash
git clone https://github.com/algojogacor/mesin-cuan.git
cd mesin-cuan

python -m venv venv
source venv/bin/activate          # Linux / Mac
# venv\Scripts\activate           # Windows

pip install -r requirements.txt
```

### Step 2 вҖ” Configure `.env`

Copy `.env.example` to `.env` and fill in your API keys:

```bash
cp .env.example .env
```

### Step 3 вҖ” Asset Bootstrap & Auth

```bash
# Download SFX packs and initialize asset folders
python tools/setup_assets.py

# One-time OAuth2 YouTube authorization (opens browser)
python setup_auth.py

# Verify all connections and dependencies
python check_requirements.py
```

## рҹ’» Usage

```bash
# Run all channels according to campaign schedule
python main.py

# Run a specific channel only
python main.py --channel ch_id_horror

# Preview campaign schedule without rendering
python main.py --preview

# Full pipeline test without uploading to GDrive
python main.py --dry-run

# Skip QC Vision for faster iteration
python main.py --skip-qc

# Refresh per-channel retention analytics
python main.py --analytics

# Generate script only (no render)
python main.py --script-only

# Review hook quality without full render
python main.py --review-hook

# Specify profile: shorts or long_form
python main.py --profile long_form

# Enable debug logging
python main.py --debug
```

## вҡЎ Quick Start (Minimal Mode)

Only need **Ollama + FFmpeg** вҖ” no API keys required:

```bash
# 1. Install Ollama and pull a model
ollama pull llama3.3:latest

# 2. Install FFmpeg
# Ubuntu: sudo apt install ffmpeg
# macOS: brew install ffmpeg
# Windows: https://ffmpeg.org/download.html

# 3. Run with Ollama only (skip footage, upload, analytics)
python main.py --channel ch_id_horror --skip-qc
```

Features that work without API keys: AI scripting (Ollama), TTS (Edge TTS free), SFX, render.
Features needing keys: B-roll footage (Pexels/Pixabay), YouTube upload, Google Drive storage.
