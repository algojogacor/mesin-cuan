## Visi & Misi

> *"Ini bukan bot. Ini arsitek konten berbasis AI yang bekerja tanpa henti вҖ” menemukan tren, menulis naskah, merender video sinematik, dan mengupload ke YouTube, semuanya otomatis."*

**Mesin Cuan Viral Architect** adalah sistem produksi konten YouTube bertenaga AI yang beroperasi penuh 24/7. Ia bukan sekadar script; ia adalah **pabrik video otonom** yang memahami algoritma, mendeteksi tren sebelum viral, dan mengeksekusi setiap frame dengan presisi sinematik.

Bayangkan memiliki tim produksi lengkap вҖ” riset tren, penulis naskah, voice-over artist, video editor, dan manajer upload вҖ” yang bekerja tanpa istirahat, tanpa gaji, tanpa keluhan. Itulah Mesin Cuan.

## вҡҷпёҸ Arsitektur 12 Engine

### рҹ“Ў Viral Loop Engine
Riset tren real-time menggabungkan **YouTube Data API v3**, **Google Trends**, dan **YouTube Search Suggestions** вҖ” diperbarui setiap 6 jam.

### рҹ§  Dual Parallel AI Scripting
**Qwen + Ollama generate naskah secara paralel**, lalu saling menilai dengan cross-provider scoring (anti-sycophancy). Skor tertinggi yang lanjut ke produksi. Fallback chain: Ollama вҶ’ Qwen вҶ’ Groq.

### рҹҺҜ Script Quality Scorer
Multi-dimensional scoring: **hook strength**, **curiosity gap**, **pacing**, **emotional impact**, **retention prediction** вҖ” naskah di bawah threshold otomatis ditulis ulang.

### рҹ”— Hook Engine
Auto-inject hook pembuka yang mempertahankan retensi penonton di 3 detik pertama вҖ” berbasis data retention analytics per channel.

### рҹ“ҡ Memory Engine
Melacak topik yang sudah diproduksi вҖ” mencegah pengulangan konten dan memastikan variasi antar video.

### рҹ”Қ Research Engine
Web research otomatis sebelum penulisan naskah вҖ” memastikan fakta akurat dan konteks terkini.

### рҹ“ә Series Engine
Auto-generate konten berseri (Part 1, 2, 3...) dengan cliffhanger logic dan call-back ke episode sebelumnya.

### рҹҢҹ Neon Visuals
Renderer video sinematik FFmpeg: **teks glowing neon**, **glassmorphism panels**, **cinematic letterbox** + vignette + color grading.

### рҹ”Ҡ Smart SFX Mixer
Sound effect otomatis berdasarkan niche: Horror вҶ’ heartbeat/thunder, Psychology вҶ’ mind-tone/focus hum, Motivation вҶ’ crowd cheer/stadium echo.

### рҹ–јпёҸ Thumbnail Intelligence
AI-driven thumbnail text & style selection berdasarkan niche dan topik вҖ” optimasi CTR.

### рҹ“Ҡ OAuth2 Analytics
Integrasi **YouTube Analytics API v2** вҖ” dashboard retensi per channel, analisis drop-off, optimasi konten.

### вҡЎ Pipeline Estimator
Prediksi ETA untuk batch render multi-channel вҖ” estimasi akurat sebelum eksekusi.

## рҹҡҖ Instalasi & Setup

### Langkah 1 вҖ” Clone & Environment

```bash
git clone https://github.com/algojogacor/mesin-cuan.git
cd mesin-cuan

python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # Linux / Mac

pip install -r requirements.txt
```

### Langkah 2 вҖ” Konfigurasi `.env`

Salin `.env.example` ke `.env` dan isi API key kamu:

```bash
cp .env.example .env
```

### Langkah 3 вҖ” Setup Assets & Auth

```bash
# Download SFX pack + buat folder assets
python tools/setup_assets.py

# OAuth2 YouTube (buka browser, login sekali)
python setup_auth.py

# Verifikasi semua koneksi
python check_requirements.py
```

## рҹ’» Cara Penggunaan

```bash
# Jalankan semua channel sesuai campaign
python main.py

# Jalankan channel tertentu saja
python main.py --channel ch_id_horror

# Preview jadwal tanpa render
python main.py --preview

# Test pipeline tanpa upload ke GDrive
python main.py --dry-run

# Skip QC Vision (lebih cepat, untuk testing)
python main.py --skip-qc

# Update dashboard analytics retensi
python main.py --analytics

# Generate naskah saja (tanpa render)
python main.py --script-only

# Review kualitas hook tanpa render penuh
python main.py --review-hook

# Pilih profil: shorts atau long_form
python main.py --profile long_form

# Aktifkan debug logging
python main.py --debug
```

## вҡЎ Quick Start (Mode Minimal)

Cuma butuh **Ollama + FFmpeg** вҖ” tanpa API key:

```bash
# 1. Install Ollama dan pull model
ollama pull llama3.3:latest

# 2. Install FFmpeg
# Ubuntu: sudo apt install ffmpeg
# macOS: brew install ffmpeg
# Windows: https://ffmpeg.org/download.html

# 3. Jalankan dengan Ollama saja (skip footage, upload, analytics)
python main.py --channel ch_id_horror --skip-qc
```

Fitur yang jalan tanpa API key: AI scripting (Ollama), TTS (Edge TTS gratis), SFX, render.
Fitur yang butuh key: B-roll footage (Pexels/Pixabay), upload YouTube, Google Drive.
